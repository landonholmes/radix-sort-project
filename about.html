About The Project:

This project is Radix Sort coded using JavaScript and HTML/CSS. It makes use of the JQuery JavaScript library, concepts such as local storage (way better than cookies) and the window.performance system for accurate algorithm timing.

The system defaults to certain input parameters upon first entry and generates an input based upon those parameters and runs radix sort on that input. The log on the right of the page displays what is happening in the JavaScript with status updates on what the input looks like before/after sorting and the time it took to sort for a specific set of parameters.

The user is given the option to change these parameters and re-run the sort using the new parameters along with a few controls on what is currently displayed in the log window. The reset button will reset the inputs back to the defaults of 2000, 2000, 100. Hitting enter inside of an input will run the sort with the current input values.

Some links to tools used:

  • Twitter Bootstrap for styling
  • JQuery for conciseness
  • Underscore.js for utility
